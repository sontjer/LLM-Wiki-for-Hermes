# Hermes IT Wiki — 总纲

## 这是什么

LLM Wiki 是从源笔记目录衍生的知识管理体系。源文件（`源文件/`）不做任何修改，所有索引、链接、分类归档都在 `Wiki/` 中进行。

## 工作流程

```
源文件/（原始笔记，不可修改）
    │
    ├→ 扫描 → 分类 → 复制到 Wiki/分类目录/
    │                   ├→ 添加 [[wikilinks]] 关联
    │                   └→ 更新 📑索引/ 目录页
    │
    └→ 用户提问 → 语义搜索 → LLM 输出 HTML
```

## 三个原则

1. **源文件不碰** — 原始笔记目录下的 .md 文件是原始资料，禁止修改
2. **AI 自动化** — 分类、建链、索引更新全部由 AI Agent 按指令执行
3. **人工可控** — 本目录下的 `指令/` 文件可随时修改，AI 下次执行时自动遵循新版规则

## 定制提示

- 修改 `01_分类规则.md` 的关键词表即可适配任意知识领域
- 修改 `04_HTML生成模板.md` 的 CSS 即可改变输出样式
- 路径常量在 `wiki_init.py` 文件开头定义
